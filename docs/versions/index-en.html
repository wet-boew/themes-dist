Changement apporté du contrôle des version à WET-BOEW et GCWeb.

En vigueur: 11 février 2019

  • Le contrôle des versions se fait selon la gestion sémantique de version
  • Le numéro de version du projet de la WET-BOEW diffère légèrement tel que définie par la gestion sémantique de version, avec l'ajout d'un suffixe numérique représentant l'architecture.
  • Le numéro de version de la WET-BOEW est maintenant indépendant du numéro de version du thème GCWeb (Canada.ca).

Cette nouvelle définition est applicable pour la version et toutes les version subséquente de :

  • WET-BOEW, version 4.0.30
  • GCWeb (Canada.ca), version 5.0

Consulter la page à propos du Contrôle des versions pour de plus amples détails.
